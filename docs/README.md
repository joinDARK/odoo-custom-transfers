## 🚀 Быстрый старт

### Для пользователей:
1. **Откройте** раздел "Справочники" → "SWIFT коды"
2. **Нажмите** "Создать"
3. **Введите** BIC код (например, `DEUTDEFF`)
4. **Сохраните** - данные загрузятся автоматически

### Для чат-бота:
```
/swift DEUTDEFF
```

### Для разработчиков:
```python
# Создание SWIFT записи
swift_record = self.env['amanat.swift'].create({
    'bic_code': 'DEUTDEFF',
})
```

## 🛠️ Установка и настройка

### Требования:
- **Odoo:** 15.0+
- **Python:** 3.8+
- **Интернет:** для API запросов
- **API ключ:** SwiftCodesAPI.com

### Быстрая установка:
```bash
# 1. Скопируйте модуль
cp -r amanat /path/to/odoo/addons/

# 2. Обновите список модулей
odoo-bin -u amanat -d your_database

# 3. Активируйте модуль
# Настройки → Приложения → Найти "SWIFT" → Установить
```

### Настройка API:
1. Получите API ключ на [SwiftCodesAPI.com](https://swiftcodesapi.com)
2. Обновите ключ в файле `models/swift.py`
3. Перезапустите Odoo сервер

## 🚨 Устранение неполадок

### Частые проблемы:

#### 1. **"API недоступен"**
- Проверьте интернет соединение
- Убедитесь в правильности API ключа
- Проверьте лимиты запросов

#### 2. **"BIC код не найден"**
- Проверьте формат кода (8 или 11 символов)
- Убедитесь в правильности написания
- Попробуйте альтернативные коды банка

#### 3. **"Медленная работа"**
- Используйте фильтры для поиска
- Избегайте массовых операций
- Проверьте производительность сервера

### Диагностика:
```bash
# Проверка логов
tail -f /var/log/odoo/odoo.log | grep amanat.swift

# Проверка API
curl -H "X-Api-Key: your_key" https://swiftcodesapi.com/v1/swifts/DEUTDEFF
```
